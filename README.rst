Rehy
====

An expirement to build hybrid app using React.

**Extremely expiremental! Expect breaking changes.**


Quick Start
-----------

.. code-block:: bash

   npm install -g rehy-cli
   mkdir new-app
   cd new-app
   rehy new
   rehy dev
   open http://localhost:8080
