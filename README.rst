Rehy
====

An expirement to build hybrid app using React.

**Extremely expiremental! Break things without notice.**


Quick Start
-----------

.. code-block:: bash

   npm install -g rehy-cli
   mkdir new-app
   cd new-app
   rehy new
   rehy dev
   open http://localhost:8080
